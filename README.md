# TW
Proyecto final de la asignatura de Teconologías Web. Universidad de Granada (UGR)

La finalidad de esta aplicación web es la gestión del calendario de vacunación de los ciudadanos de la comunidad autónoma de Andalucia.

El sistema gestionará la información relativa a los usuarios del sistema, las vacunas disponibles, el calendario de vacunación, etc. Estos datos serán almacenados en una base de datos a la cual accederemos medainte claúsulas SQL que nos permitirán gestionar dichos datos. Habrá diferentes tipos de usuarios, entre los que encontramos los administradores, los sanitarios y los pacientes, y un usuario podrá realizar diferentes operaciones según el tipo de usuario que sea.
